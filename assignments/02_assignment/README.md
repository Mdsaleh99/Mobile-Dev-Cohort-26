## 📂 Project Structure

```text
 assignments/02_assignment/
 ├── assets/                 # App assets (Videos, Icons, Images)
 ├── src/
 │   ├── app/
 │   │   ├── _layout.tsx     # Root layout & Navigation
 │   │   ├── components/     # Screen Components
 │   │   │   ├── NoteEditorScreen.tsx
 │   │   │   └── NotesListingScreen.tsx

 └── tsconfig.json           # TS Configuration
```
